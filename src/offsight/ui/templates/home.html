OffSight™ - Regulatory Intelligence

Prototype system for monitoring and analysing UK offshore wind regulatory changes.

What this system does: OffSight monitors configured regulatory sources, stores versioned documents, detects changes between versions, and uses a local AI model (Ollama) to generate short summaries and impact categories. Human reviewers validate or correct these suggestions to create an auditable decision trail.

End-to-end workflow

  1. Sources are configured on the Sources page.
  2. Scraper jobs fetch HTML content and store versioned documents.
  3. Change detection compares versions and creates RegulationChange records.
  4. AI summarisation runs locally via Ollama to suggest a summary and impact category.
  5. Humans review each change on the Changes page and record validation decisions.

Use the navigation links above to explore the system: start with Sources to see what is monitored, then review detected changes on the Changes page.

OffSight™ - Regulatory Intelligence

Prototype system for monitoring and analysing regulatory changes in UK offshore wind guidance.

What this system does: OffSight monitors configured regulatory sources, stores versioned documents, detects changes between versions, and uses a local AI model (Ollama) to summarise the impact of those changes. Human reviewers then validate or correct the AI suggestions so you have an auditable view of what changed and why it matters.

End-to-end workflow

  1. Sources are scraped and stored as versioned regulatory documents.
  2. Change detection compares versions and creates RegulationChange records with diffs.
  3. AI generates a short summary and impact category for each change.
  4. Humans approve, correct, or reject suggestions, creating a validation trail.
