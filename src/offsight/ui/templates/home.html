OffSight™ - Regulatory Intelligence

Prototype system for monitoring and analysing regulatory changes in UK offshore wind guidance.

What this system does: OffSight monitors configured regulatory sources, stores versioned documents, detects changes between versions, and uses a local AI model (Ollama) to summarise the impact of those changes. Human reviewers then validate or correct the AI suggestions so you have an auditable view of what changed and why it matters.

End-to-end workflow

  1. Sources are scraped and stored as versioned regulatory documents.
  2. Change detection compares versions and creates RegulationChange records with diffs.
  3. AI generates a short summary and impact category for each change.
  4. Humans approve, correct, or reject suggestions, creating a validation trail.

Use the navigation links above to configure Sources and review detected Changes. This UI is intentionally minimal and optimised for the exam demo.
